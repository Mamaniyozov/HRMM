# HRMM Frontend

Bu papka alohida frontend prototipi uchun tayyorlandi.

## Ishga tushirish

1. Backendni `C:\Users\maman\Documents\HRMM\hrmm_backend` ichidan ishga tushiring:
   `.\venv\Scripts\python.exe manage.py runserver`
2. Frontend papkaga kiring:
   `C:\Users\maman\Documents\HRMM\hrmm_frontend`
3. Oddiy local server oching:
   `python -m http.server 5500`
4. Browserda oching:
   `http://127.0.0.1:5500`

## Eslatma

- Frontend local serverdan ochilishi uchun backendga yengil CORS middleware qoshildi.
- Dizayn IT staffing panel uchun rasmiy va boshqaruv uslubida tayyorlangan.
- Keyinchalik shu layoutni React yoki Vite loyihasiga oson kochirish mumkin.
